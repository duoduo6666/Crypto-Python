# Crypto
## 支持的加密算法
+ SHA-1
    + SHA1
    + SHA1s
+ Base64
    + Base64_encode
    + Base64_decode

## 提交类型

| 提交类型    | 标题      | 描述                                       | Emoji | 版本    |
|:-----------:|-----------|--------------------------------------------|:-----:|---------|
| `feat`      | 功能	  | 添加新功能                                 |  ✨   | `minor` |
| `fix`       | Bug修复	  | 修复Bug                                    |  🐛   | `patch` |
| `docs`      | 文档	  | 文档修改                                   |  📚   | -       |
| `style`     | 样式      | 不影响代码含义的更改 (空格, 格式, 注释等)  |  💎   | -       |
| `refactor`  | 代码重构  | 既不修复错误也不添加功能的代码更改         |  📦   | -       |
| `perf`      | 性能改进  | 提高性能的代码更改                         |  🚀   | `patch` |
| `test`      | 测试      | 添加缺失的测试或纠正现有的测试             |  🚨   | -       |
| `revert`    | 还原      | 还原之前的提交                             |  🗑   | -       |
